## 📸 Overview

A clean, professional portfolio built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies. Designed with an editorial aesthetic using the Fraunces serif typeface and a warm cream/forest green color palette.

---

## ✨ Features

- **Smooth scroll navigation** with sticky navbar on scroll
- **Animated hero section** with availability status indicator
- **Accordion project cards** — click to expand each project
- **Scroll-reveal animations** on all sections
- **Custom animated cursor** with ring follow effect
- **Fully responsive** — optimized for mobile and desktop
- **Single HTML file** — zero build step, zero dependencies
